## Taalopzet

Deze module blijft Nederlandstalig als brontaal en bevat daarnaast een Engelse vertaling in `i18n/en_US.po`.
Daardoor past de interface zich aan op basis van de actieve Odoo-taal.

## Functies

- Projectgroepen op producten
- Toegestane projectgroepen op prijslijsten
- Verboden producten op prijslijsten
- Niet toestaan op prijslijsten op productniveau
- Automatische cleanup van niet-toegestane verkoopregels
- Ondersteuning voor optionele producten
